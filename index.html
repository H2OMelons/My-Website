Projects

Muse.k

August 2017-Current

Muse.k is a Chrome extension I worked on that allows the user to play Youtube videos in the background with a playlist feature so the user can save videos they like for future sessions.
The extension uses Chrome's storage API to save playlists, Youtube's Data API to implement a search mechanism and playlist importing, and Chrome's identity API for user authentication to access and import their private playlists.

JavaScript, HTML, CSS, Chrome API, IFrame API

DejaPhoto

April 2017-June 2017

DejaPhoto is an Android app that lets people reminisce on the photos that they took in the past while also being able to share these memories with their friends.
DejaPhoto was a collaborative team project that used Agile Software Development methods to develop the final product and Jira to track issues and develop new features.
I implemented the backend of the app by using Firebase for photo sharing and friend lists while also writing the tests for most of the code.

Java, Android Studio, Jira, Google Maps API, Firebase API
